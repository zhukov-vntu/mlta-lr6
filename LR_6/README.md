# How to use?

## 1. Clone the repo

```shell
git clone https://github.com/zhukov-vntu/mlta-lr6.git
cd task1
```
## 2. Install dependencies

Create virtual environment:

python -m venv .venv

### Activate venv

**Linux / macOS**
source .venv/bin/activate

**Windows (PowerShell)**
.venv\Scripts\Activate.ps1

**Windows (CMD)**
.venv\Scripts\activate.bat

Install dependencies:

pip install -r requirements.txt

## 3. Run main.py

```shell
python main.py
```
